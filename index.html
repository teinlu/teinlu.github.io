GLARE: Geometry-based Local Anomaly Recognition and Encoding for Few-Shot 3D Industrial Anomaly Detection [Paper]
Tao Lu, et al.
Under Review Manuscript

GLARE detects 3D industrial defects using only point cloud geometry — no RGB, no large pretrained encoders. A compact 12-dimensional geometric descriptor (7 shape-ratio + 5 local height statistics) built from 7,000+ ablation configurations feeds a normal-feature memory bank; anomalies are localized by maximum feature deviation. A single neighborhood scale (k = 16) matches five-scale ensembles at 4× lower latency. Density-Aware Memory Scoring (DAMS), a parameter-free percentile-weighted pooling operator, corrects for variable point density and yields a consistent +2.2 pp gain over standard max-pooling. Evaluated under an 8-shot depth-only protocol, GLARE reaches 66.1% AUROC on Real3D-AD and 75.1% on MVTec3D-AD, narrowing the geometry-to-RGB-D accuracy gap to 4.3 pp on Real3D-AD. Each sample processes in 31 ms (26× faster than FPFH).

SurfConv: Treating Organized Point Clouds as Geometry Images for Industrial Anomaly Detection [Paper]
Tao Lu, et al.
Under Review Manuscript

Structured-light depth scanners produce organized point clouds (H × W grids where grid adjacency directly encodes surface adjacency), yet leading 3D anomaly detection methods discard this structure by converting scans to unordered sets that require Open3D, MinkowskiEngine, or PointTransformer. SurfConv shows this is avoidable for industrial structured-light data: organized point clouds are treated as geometry images — standard 2D tensors processable by frozen ImageNet backbones. The key component is FDGE (Finite-Difference Geometry Encoding): a pure-PyTorch geometry parameterization that computes 5-channel surface descriptors (normals n, curvature proxies H, ˜K) at <10 ms per image with zero external 3D library dependencies. These geometry features are fused with RGB via a dual-stream PatchCore memory bank. On MVTec3D-AD (10 categories), SurfConv achieves 98.3% Pixel-AUROC (3-seed mean) — competitive with or above 3D-library-dependent methods (PO3AD: 97.5%, Shape-Guided: 97.4%, BTF: 96.1%) — with zero external 3D library dependencies. FDGE geometry encoding improves I-AUROC by +8.4 pp over RGB-only (3-seed per-category mean; p = 0.009**, exploratory). On Real3D-AD (12 categories, 4–9 training samples), SurfConv achieves 90.5% P-AUROC (3-seed mean), competitive with the strongest prior 3D baseline (R3D-AD: 90.3%) while requiring no 3D libraries.
